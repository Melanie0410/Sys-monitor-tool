# Sys Monitor Tool

Este proyecto contiene una serie de **scripts en Bash** para:
- **Monitorear recursos del sistema** (`resource_monitor.sh`)
- **Automatizar backups** (`backup_manager.sh`)
- **Analizar logs del sistema** (`log_analyzer.sh`)
